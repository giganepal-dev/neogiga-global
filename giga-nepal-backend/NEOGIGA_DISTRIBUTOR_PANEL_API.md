# NeoGiga Distributor Panel API

Existing protected distributor APIs:

- Dashboard
- Profile
- Territories
- Leads list/create
- Customers
- Orders
- Commissions
- Payouts
- Downlines

Security:

- Requires `api.token`
- Requires distributor permissions
- Distributor must only see assigned territory summaries.
