# Global Schema Guide

Implement these schema types by page:

- Home: Organization, WebSite, SearchAction, FAQ
- Product: Product, Offer, Breadcrumb, FAQ, ShippingDetails, MerchantReturnPolicy
- Category: CollectionPage, Breadcrumb, FAQ
- Manufacturer/Brand: Brand or Organization, CollectionPage, Breadcrumb
- Seller: Store, LocalBusiness, AggregateRating
- LMS: Course, VideoObject, Breadcrumb
- Blog/Knowledge: Article, FAQ
- AI: SoftwareApplication
- RFQ: WebPage, Breadcrumb

Schema must use real stock, price, rating and return policy data only.
