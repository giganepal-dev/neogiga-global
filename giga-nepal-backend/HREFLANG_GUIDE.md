# Hreflang Guide

Use one canonical product and generate alternates per locale prefix.

Required alternates:

- `x-default`
- `en`
- `en-IN`
- `en-NP`
- `en-BD`
- `en-MM`
- `en-AU`
- `en-US`
- `en-CA`
- `en-GB`
- `en-AE`
- `en-QA`
- `en-SA`
- `de-DE`
- `fr-FR`
- `es-ES`
- `it-IT`
- `pt-BR`

The new `GlobalSeoI18nService` can generate prefix-based hreflang clusters from `config/neogiga_global.php`.
