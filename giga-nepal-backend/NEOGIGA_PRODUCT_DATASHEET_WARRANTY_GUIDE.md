# NeoGiga Product Datasheet Warranty Guide

Tables added:

- `product_datasheets`
- `product_certificates`
- `product_manuals`
- `product_warranties`

Public APIs:

- `GET /api/v1/products/{product}/datasheets`
- `GET /api/v1/products/{product}/warranty`

Upload and admin review APIs remain the next step. File type and size validation must be enforced when uploads are enabled.
