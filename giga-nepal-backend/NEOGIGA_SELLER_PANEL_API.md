# NeoGiga Seller Panel API

Existing protected seller APIs:

- Dashboard overview and summaries
- Profile read/update
- Marketplace approvals
- Product list/create/show/update/submit-review
- Inventory list/adjust
- Orders and status updates
- Payouts placeholder
- Performance
- Support tickets

Security:

- Requires `api.token`
- Requires seller permissions
- Seller policies restrict own records

Remaining work: document upload, datasheet, variant/spec/warranty management endpoints.
