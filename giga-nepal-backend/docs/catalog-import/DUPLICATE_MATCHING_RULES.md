# Duplicate Matching Rules

1. Verified GTIN, UPC, EAN, or ISBN.
2. Exact canonical brand/manufacturer and normalized MPN.
3. Exact supplier and source product ID.
4. Exact supplier SKU.
5. Verified canonical URL.
6. Manual review only for normalized model/title similarity.

Independent products must never become variants because their names are similar. Duplicate candidates are retained as review tasks with source evidence; no automated merge or deletion occurs.
