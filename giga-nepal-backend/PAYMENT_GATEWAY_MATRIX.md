# Payment Gateway Matrix

| Market | Gateways |
|---|---|
| India | Razorpay, PayU, PhonePe, UPI, Cash |
| Nepal | eSewa, Khalti, FonePay, IME Pay, COD |
| Bangladesh | bKash, Nagad, Rocket |
| Australia | Stripe, PayPal, Apple Pay, Google Pay |
| Global | Stripe, PayPal, Wire, Bank Transfer |

Only local gateways should be visible at checkout after the payment provider layer is enabled.
