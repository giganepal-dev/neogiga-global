# Global Redirect Policy

- Old URLs remain valid.
- Locale-prefixed URLs are additive and backward-compatible.
- Geo routing is recommendation-first unless `NEOGIGA_GEO_RECOMMENDATION_REDIRECT=true`.
- Never force users away from a manually selected country.
- Store preference with existing marketplace preference cookie.
- Canonical should point to the active locale URL once localized sitemap rollout is enabled.
